# keyboard-stuff

replace `~/.config/karabiner/karabiner.json` with the thing in here to remap

## how to update

`st ~/.config/karabiner/karabiner.json`

change the thing and save

test.

If good also update repo and commit

cd into repo and
`st ./karabiner.json`
